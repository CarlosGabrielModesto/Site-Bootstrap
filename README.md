# BOOTSTRAP

É um **framework front-end** que fornece estruturas de CSS para a criação de sites e aplicações responsivas de forma rápida e simples. Isso significa que seu site ou aplicativo terá uma ótima aparência e funcionará bem em qualquer dispositivo, sem a necessidade de escrever código adicional ou consultas de mídia.
Ele também inclui um sistema de **GRID** que permite criar layouts complexos de várias colunas com facilidade.

Você pode começar com o Bootstrap incluindo os arquivos CSS e JavaScript necessários no cabeçalho do seu arquivo HTML e, em seguida, adicionando as classes apropriadas aos seus elementos HTML.

Para aprender mais sobre o Bootstrap e seus recursos acesse o site oficial em https://getbootstrap.com/docs/4.5/getting-started/introduction/

Neste repositório trago um exemplo basico de um site utilizando bootstrap.
